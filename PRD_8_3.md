## 2. Theorie-inhoud (uit bron p. 55)

### Kernregel

> Bij een vergelijking kun je denken aan een balans.
> Je kunt de vergelijking oplossen door links en rechts van het '= teken' **hetzelfde** te doen.

### Voorbeeld uit het boek (p. 55)

De balans hoort bij de vergelijking `4a + 1 = 2a + 5`. Los de vergelijking op met een balans.

| Stap | Actie | Vergelijking |
|------|-------|-------------|
| Balans 1 (start) | — | `4a + 1 = 2a + 5` |
| Actie 1 | Links en rechts 1 knikker weghalen | ↓ |
| Balans 2 | — | `4a = 2a + 4` |
| Actie 2 | Links en rechts twee zakjes weghalen | ↓ |
| Balans 3 | — | `2a = 4` |
| Conclusie | In twee zakjes zitten vier knikkers, dus in één zakje zitten twee knikkers | `a = 2` |

### Verschil met §8.2

| §8.2 | §8.3 |
|------|------|
| Letters alleen aan één kant (`na + c = d`) | Letters aan **beide** kanten (`pa + c = qa + d`) |
| Alleen getallen weghalen | Eerst zakjes (letters) weghalen, dan getallen |

## 3. Digibord Mode — Lesflow §8.3

### Stap 1: Slides tonen
- App laadt `/slides/8_3.pdf` in de in-app PDF viewer.
- Docent bladert door slides (vergelijkingen met letters aan beide kanten).

### Stap 2: Theorie interactief
- Balance Lab in digibord-modus toont `4a + 1 = 2a + 5`.
- Docent doorloopt het voorbeeld stapsgewijs:
  1. Balans toont 4 zakjes + 1 knikker links, 2 zakjes + 5 knikkers rechts.
  2. Klikt "Volgende stap" → 1 knikker verdwijnt aan beide kanten → `4a = 2a + 4`.
  3. Klikt "Volgende stap" → 2 zakjes verdwijnen aan beide kanten → `2a = 4`.
  4. Conclusie: `a = 2`.

### Stap 3: Klassikale inoefenopgave
- **Gekozen opgave: Opgave 15** (p. 54)
- Context: Balans met drie stappen (❶❷❸). Vergelijking `4g + 3 = 2g + 17`.
  - a. Hoeveel kazen zijn er links en rechts weggehaald om de situatie van balans ❷ te krijgen?
  - b. Bij balans ❷ wordt links en rechts het gewicht van 3 kg weggehaald. Teken de gewichten die bij balans ❸ aan de rechterkant overblijven.
  - c. Scott heeft bij balansen ❶ en ❷ de vergelijkingen opgeschreven. Leg uit hoe hij aan deze vergelijking komt.
  - d. Schrijf de vergelijking op die bij balans ❸ hoort.
  - e. Bereken het gewicht van één kaas.
- Docent demonstreert in Balance Lab op digibord.

### Stap 4: Start zelfstandig werk
- Docent drukt op "Start zelfstandig werk" → timer start (45 min).

### Balance Lab rol in §8.3

In §8.3 wordt de Balance Lab de primaire oplosmethode:

1. **Voorinvullen:** bij elke opgave met 🔬 wordt de vergelijking voorgevuld in Balance Lab.
2. **Stapsgewijs oplossen:** leerling voert acties uit ("Haal van beide kanten 2a weg" → "Haal van beide kanten 4 weg").
3. **Stappenlog:** elke actie wordt gelogd:
   ```
   Stap 1: beide kanten −1       → 4a = 2a + 4
   Stap 2: beide kanten −2a      → 2a = 4
   Stap 3: beide kanten ÷2       → a = 2
   ```
4. **Coach Hints** (volgorde conform boek-aanpak):
   - Hint 1: "Haal eerst dezelfde zakjes (letters) aan beide kanten weg."
   - Hint 2: "Haal nu de losse knikkers (getallen) aan beide kanten weg."
   - Hint 3: "Deel door het aantal zakjes (de coëfficiënt)."
   - Hint 4: "Controleer je antwoord door in te vullen."
5. **Undo/Redo:** volledige rollback bij verkeerde stap.

## 6. Feedback Tags §8.3

| Error Tag | Wanneer | Voorbeeldfout |
|-----------|---------|--------------|
| `BALANCE_ONE_SIDE` | Alleen aan één kant iets weggehaald | `4a + 1 = 2a + 5` → `4a = 2a + 5` (alleen links −1) |
| `SIGN_ERROR` | Verkeerd teken bij weghalen | `−2a` aan linkerkant maar `+2a` aan rechterkant |
| `DIVIDE_COEFF_MISS` | Vergeten te delen door coëfficiënt | `2a = 4` → antwoord `a = 4` |
| `CHECK_FAILED` | Controle niet uitgevoerd of controle klopt niet | Geen controle-stap of verkeerd ingevuld |
| `ARITHMETIC_ERROR` | Rekenfout | `4 − 1 = 2` |
| `WRONG_TERM_REMOVED` | Verkeerde term als eerste weggehaald | Bij `4a + 1 = 2a + 5`: eerst zakjes ipv knikkers (niet fout maar suboptimaal — zachte hint) |
| `INCOMPLETE_SOLVE` | Niet doorgerekend tot `a = ...` | Gestopt bij `2a = 4` zonder te delen |

## 7. Leerdoelencheck §8.3

**Trigger:** Leerling heeft alle opgaven van gekozen route afgerond.

| Vraag (exact uit bron p. 56) | Gekoppeld aan |
|------------------------------|--------------|
| De balans hoort bij de vergelijking `3a + 4 = a + 14`. Los de vergelijking op met de balans. | → opdracht E5 en E6 |

### Verwachte oplossingsstappen (in Balance Lab)
1. Beide kanten −a → `2a + 4 = 14`
2. Beide kanten −4 → `2a = 10`
3. Beide kanten ÷2 → `a = 5`
4. Controle: links `3×5 + 4 = 19`, rechts `5 + 14 = 19` ✓

### Beoordeling
- **Correct** (`a = 5` + controle) → Doorgaan naar E5/E6 of Generator.
- **Incorrect** → Fout-feedback via error-tag + terugverwijzing theorie §8.3.

## 9. Oneindige Generator §8.3

**Generator-ID:** `BALANCE_TWO_SIDES`

### Vergelijkingstype
`p·a + c = q·a + d` (letters aan beide kanten van het = teken)

### Generatorlogica

| Parameter | Herhaal-modus | Verdiep-modus |
|-----------|--------------|---------------|
| Coëfficiënt links (p) | 2–6 | 2–10 |
| Coëfficiënt rechts (q) | 1–5 (q < p) | 1–9 (q ≠ p) |
| Constante links (c) | 1–15 (positief) | 0–20 (positief) |
| Constante rechts (d) | 1–15 (positief, d > c) | 0–20 (positief) |
| Variabele | a | a–z (random) |
| Visuele balans | altijd getoond | altijd getoond |
| Balance Lab knop | altijd beschikbaar | altijd beschikbaar |
| Oplossing a | ∈ ℕ, a ≥ 1 | ∈ ℕ, a ≥ 1 |

### Constraints
- `(d − c)` moet deelbaar zijn door `(p − q)` zodat `a` een geheel positief getal is.
- In herhaal-modus: `p > q` (zodat letters altijd naar links gaan — simpeler).
- In verdiep-modus: `q > p` is ook mogelijk (letters naar rechts verplaatsen).

### Voorbeeldgeneratie

| Modus | Vergelijking | Stap 1 | Stap 2 | Stap 3 | Antwoord |
|-------|-------------|--------|--------|--------|----------|
| Herhaal | `5a + 3 = 2a + 12` | −2a → `3a + 3 = 12` | −3 → `3a = 9` | ÷3 | `a = 3` |
| Herhaal | `4a + 2 = a + 14` | −a → `3a + 2 = 14` | −2 → `3a = 12` | ÷3 | `a = 4` |
| Verdiep | `3b + 8 = 7b + 0` | −3b → `8 = 4b` | ÷4 | — | `b = 2` |
| Verdiep | `2x + 15 = 5x + 3` | −2x → `15 = 3x + 3` | −3 → `12 = 3x` | ÷3 | `x = 4` |

### Feedback bij generator
- Dezelfde error-tags als §6 hierboven.
- Elke opgave biedt "Open in Balance Lab" knop.
- Balance Lab toont visuele balans met zakjes aan beide kanten.
- Na 3 opeenvolgende fouten → hint + theorie-popup.
- Na 5 correcte op rij → "Je kunt vergelijkingen oplossen met de balans!" + optie stoppen/doorgaan.
